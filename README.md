# SenSokScript 🎮

A customized Blox Fruits script based on Xynapse, modified and maintained by **chhinhlongdev (CodeKley)**.

## 📋 Features

- Auto Farm Level
- Quest System
- Combat Module
- Gun Module
- Optimization Tools
- Server Management
- Custom UI Library

## 🚀 Installation

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/chhinhlongdev/SenSokScript/main/Utils/Module.lua"))()
```
