# SenSokScript Assets

## Banner Image Setup

អ្នកត្រូវដាក់រូបភាពថ្មីរបស់អ្នកនៅទីនេះ:

### ឈ្មោះឯកសារ: `Banner.png`
- រូបភាពនេះនឹងត្រូវបានប្រើជា banner សម្រាប់ UI
- ទំហំដែលសមស្រប: 1024x576 pixels ឬធំជាងនេះ
- Format: PNG (recommended) ឬ JPG

### របៀបដាក់រូបភាព:
1. រក្សាទុករូបភាពរបស់អ្នកជា `Banner.png`
2. ដាក់វានៅក្នុង folder `Assests/`
3. Upload ទៅ GitHub repository របស់អ្នក: `https://github.com/chhinhlongdev/SenSokScript`

### Path ពេញលេញ:
```
Assests/Banner.png
```

### URL នៅលើ GitHub:
```
https://raw.githubusercontent.com/chhinhlongdev/SenSokScript/main/Assests/Banner.png
```

**Note:** រូបភាពដែលអ្នកបានផ្តល់ឱ្យ (anime character with purple eyes and wings) គួរតែត្រូវបាន save ជា `Banner.png` នៅក្នុង folder នេះ។
